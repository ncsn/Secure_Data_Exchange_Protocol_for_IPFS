Get File

Enter CID¹ (shared privately by the owner). The system uses it to locate and authenticate the owner via the 4-step Enhanced Privacy Protocol.

CID¹ (H(OBJ)) — used for retrieval: —
1
WANT_HAVE(CID³)
A sends WANT_HAVE using CID³ to locate owner B
2
PRIVACY_CHALLENGE
B proves ownership by signing CID² — sig(H(H(OBJ)))
3
PRIVACY_RESPONSE
A encrypts CID¹ + session key K with B's public key (ECIES)
4
PRIVACY_BLOCK
B sends AESK(OBJ), A decrypts and verifies H(OBJ) = CID¹
